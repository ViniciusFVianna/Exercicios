# 📚 Exercícios de Programação & Lógica

Este repositório é dedicado ao armazenamento de exercícios, desafios de algoritmos e resoluções de problemas acadêmicos desenvolvidos durante a graduação em **Engenharia de Software**.

## 🎯 Objetivo

Centralizar estudos e práticas sobre fundamentos da computação, estruturas de dados e lógica matemática, servindo como material de consulta e acompanhamento de evolução técnica.

## 📂 Organização do Repositório

Os exercícios estão divididos por domínios de conhecimento:

* **Lógica Proposicional:** Resoluções de tabelas verdade, validade de argumentos e conectivos lógicos.
* **Aritmética Computacional:** Conversão de bases (Binário, Hexadecimal, Octal), operações com sinal-magnitude e representação IEEE 754.
* **Algoritmos:** Problemas de lógica de programação resolvidos em [Dart/C/Python - ajuste as linguagens].
* **Matemática Discreta:** Exercícios sobre conjuntos, indução e grafos.

## 🛠️ Tecnologias Utilizadas

A maioria das resoluções utiliza:
- **Dart/Flutter** (para exercícios de lógica aplicada).
- **Markdown/LaTeX** (para documentação de fórmulas e provas lógicas).
- **Scripts de Automação** (para verificação de cálculos binários).

## 🚀 Como navegar

Cada pasta possui seu próprio arquivo de descrição ou comentários no código explicando o enunciado do problema e a lógica da solução aplicada.

1. Navegue até a pasta desejada.
2. Abra o arquivo `.dart` ou `.txt` para visualizar a implementação.
3. No caso de exercícios de lógica, verifique os arquivos `.md` para as tabelas e provas.
